Basic Data Types


    There are several types of data (datatypes) you will use:

        // STRING | Text and digits that won't work in math functions
            // always in quotes either " " or ' '
            'My Name' is a string

        // NUMBER | Digits used for math
            // No quotes
            1999 is a Number 

            // If you put digits in quotes it becomes a string
            '1999' is a String
        
            // What's the difference?
            
                // Numbers, add together mathmatically
                2 + 2 = 4 
                
                // Strings, concactenate into a new string
                '2' + '2' = '22'

        // BOOLEAN | only two values: true or false <- always lowercase

            // Your attendance is a Boolean
            student_is_in_class = true;
            
            //unless you are somewhere else
            student_is_in_class = false;

Variable Types


    We declare variables in two ways:

        // CONST | Variables who's value cannot be reassigned
        const the_greeting = "Hello";

        // LET | Varibales who's value can be reassigned:
        let the_object = "World";

            // Using Variables
            
                // declare your variable
                let first_name = 'Sean'
                
                // Now you can use the variable (no need for 'let' that's only for declaring)
                console.log(first_name) 

                // you can also change your variables
                first_name = 'Tucker'

                console.log(first_name)
                
                // If you try to reassign a CONST it will throw an error
                // check the console to see the error
                the_greeting = 'hi'

If Statements

    

    // we can take action based on the comparisons
    // the structure is: if () { }
    // () where the comparison happens
    // {} the code that runs if the comparison is true
        
        let big = 100;
        let small = 10;

        if (big > small){
            console.log("It's True");
        }

        
    
    // but what if the results is false
    // we add an ELSE

        let big = 100;
        let small = 10;

        if (big == small){
            console.log("It's True");
        } else {
            console.log("It's False");
        }
